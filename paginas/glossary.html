Glosario Técnico

Inglés Español Definición
1. Accessibility Accesibilidad Diseño de productos digitales para que puedan ser usados por cualquier persona, incluyendo aquellas con discapacidades.
2. Adaptability Adaptabilidad Capacidad de un profesional para ajustarse rápidamente a los cambios y nuevas tecnologías del mercado laboral.
3. Assets Recursos Digitales Archivos individuales como imágenes, audios o modelos 3D que forman parte de un proyecto de software o videojuego.
4. Backend Desarrollo del Lado del Servidor Capa interna de una aplicación encargada de la lógica, la seguridad y la gestión de la base de datos.
5. Cloud Computing Computación en la Nube Servicios informáticos que se ofrecen a través de Internet, como el almacenamiento de datos y servidores remotos.
6. Code Código Conjunto de instrucciones escritas en un lenguaje de programación que una computadora puede ejecutar.
7. Component Componente Elemento de diseño o código reutilizable dentro de una interfaz, como un botón o una barra de navegación.
8. Copyright Derechos de Autor Normas legales que protegen la propiedad intelectual y las creaciones de los desarrolladores y diseñadores.
9. Data Integrity Integridad de Datos Precisión, consistencia y confiabilidad de la información almacenada en una base de datos.
10. Database Base de Datos Sistema organizado para almacenar, gestionar y recuperar grandes volúmenes de información de forma eficiente.
11. Deployment Despliegue Proceso de publicar una aplicación o sitio web en un servidor de producción para que el público pueda usarlo.
12. Ethics Ética Principios morales que guían el comportamiento responsable en el uso de la tecnología y los datos de usuarios.
13. Figma Figma Herramienta en la nube utilizada para diseñar prototipos y sistemas de interfaces gráficas de forma colaborativa.
14. Frontend Desarrollo del Lado del Cliente Parte visual e interactiva de un sitio web con la que el usuario interactúa directamente.
15. Gameplay Jugabilidad Conjunto de reglas, mecánicas y experiencias de interacción de un usuario dentro de un videojuego.
16. HTML HyperText Markup Language Lenguaje de marcado estándar utilizado para estructurar el contenido de las páginas web.
17. Innovation Innovación Creación o mejora de productos, procesos y servicios mediante la aplicación de nuevas ideas y tecnologías.
18. Interface System Sistema de Interfaces Conjunto lógico y estructurado de componentes visuales que mantienen la consistencia en una plataforma.
19. Layout Disposición Estructura o distribución física en la que se organizan los elementos visuales dentro de una pantalla.
20. Mechanics Mecánicas Reglas y acciones que definen cómo funciona el sistema de juego y qué puede hacer el jugador.
21. Multimedia Multimedia Combinación de múltiples medios de expresión como textos, imágenes, videos y audios para transmitir información.
22. Networking Red de Contactos Construcción de relaciones profesionales para descubrir oportunidades laborales y colaboraciones en la industria.
23. Prototyping Prototipado Creación de modelos preliminares o bocetos interactivos de una aplicación para probar su funcionamiento antes de programarla.
24. Relational Model Modelo Relacional Enfoque de organización de bases de datos que organiza la información en tablas compuestas por filas y columnas.
25. Software Programa Informático Componentes lógicos e intangibles de un sistema informático que permiten realizar tareas específicas.
26. Soft Skills Habilidades Blandas Atributos personales como el trabajo en equipo, la comunicación y la empatía, altamente valorados por las empresas.
27. UI User Interface / Interfaz de Usuario Entorno visual y gráfico que permite la interacción directa entre una persona y un dispositivo o programa.
28. UX User Experience / Experiencia de Usuario Sensación, percepción y grado de satisfacción que experimenta una persona al usar un producto digital.
29. Visual Hierarchy Jerarquía Visual Técnica de diseño que consiste en dar mayor importancia a ciertos elementos para guiar de manera intuitiva el ojo del usuario.
30. Web Development Desarrollo Web Proceso técnico que involucra la creación, diseño, programación y mantenimiento de sitios y aplicaciones de internet.
